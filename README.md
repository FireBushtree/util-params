# envcloud-utils-params

用来获取项目中的`tenantId`, `token`, `userId`等信息

## Install

```
yarn add envcloud-utils-params
```

or

```
npm i envcloud-utils-params
```
## Getting Started

Install dependencies,

```bash
$ npm i
```

Start the dev server,

```bash
$ npm start
```

Build documentation,

```bash
$ npm run docs:build
```

Build library via `father-build`,

```bash
$ npm run build
```
